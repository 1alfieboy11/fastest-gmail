Adjust Popup Width and Height Popup width is px.

Minimum popup width is 330px.

Popup height is px.

Minimum popup height is 400px.
Time Interval to Update the Badge Icon Check Gmail Inbox every seconds.

Minimum time interval is 15 seconds.
Open Gmail Open Gmail in (inbox.google.com)

Check this box if you want to open Gmail in (inbox.google.com) when clicking on the Gmail button in toolbar-popup.
Add Gmail Labels Enter the name(s) of your Gmail Label(s) that you want to track (for the primary account)

Labels are comma (,) separated. i.e. label1, label2, label3, ...
Desktop Notifications Check this box if you want to show desktop notification popup for new emails

The popup will close automatically after few seconds.
Maximum Number of Desktop Notifications Enter the desired maximum number of desktop notifications you want to see

The minimum number is 1. If you want to disable Desktop Notifications, see the above option.
Beep-sound for New Emails Check this box if you want to hear a beep sound when a new email arrives

The beep sound lasts 1 second.
Badge-icon Background Color Pick a color for badge-icon background

Default badge background color is red.
